# BASH code

	```bash
		echo "Bash is cool"
	```

## requirements
	following are the requirements to run the above command. Although you will get it pre-installed. So, just check it in your package manager.
	**For Arch**
			sudo pacman -S bash
			sudo pacman -S echo
	**For Debian**
			sudo apt upgrade
			sudo apt install bash
			sudo apt install echo
